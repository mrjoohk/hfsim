# 제안 기술의 WBS Lv2 기술에 대한 주요 기능(WBS Lv3)
- 기능 분류 (WBSLv2)
 * 기능 항목(WBS Lv3):
  ** 주요 기능(WBS Lv4)

- 전장 디지털트윈 환경 및 월드모델 기반 전술 상황 추론
 * 물리기반 임무환경 모델링
  ** 고속 무인기·유인기 비행체 6-DOF 동역학 모델 기반 기동·자세·속도 물리 시뮬레이션
  ** 지상 방공포·SAM·기동위협 탐지 범위, 교전 반응, 킬체인 파라미터 동적 모델링
  ** 적 전투기·공대공 미사일 궤적, 교전 거리·속도·회피 기동 파라미터 시뮬레이션
  ** 고정·이동 임무 표적 위치·우선순위·방호 상태 동적 모델링 및 Ground Truth 라벨 자동 생성
  ** DEM 기반 지형 고도·기상 조건·전자전(EW) 환경 통합 시뮬레이션 및 병렬 에피소드 롤아웃 인프라 구성
  ** 5가지 이질 물리모델 단일 통합, 고충실도 전장 환경 구축
 * 월드모델 기반 예측/생성
  ** RSSM 인코더·디코더를 통한 전장 관측 데이터 압축 잠재 공간 표현 학습
  ** 학습된 전이 모델 기반 n-step 선행 위협 위치·행동 궤적 예측 및 불확실성 정량화
  ** 월드모델 내 가상 롤아웃을 통한 Dreamer 정책 학습 가속화 인터페이스 제공
  ** 극한 조건 희귀 시나리오 자동 생성 및 KPP/KSP 정량 평가 피드백 연계

- 최적 임무 수립 AI
 * 공대지/공대공 협업 자율 임무 수립
  ** 다차원 전장 정보를 기반으로 아군/적군 자산 상태, 위협/환경 정보, 월드모델 기반 미래 상태를 예측하여 전장상황 인지 및 위협도 추론
  ** 위협도/생존성/자산 가용성 등을 종합하여 다수 무인기 대상 임무 배분 및 수행 순서 동적 최적화
  ** 임무 할당 결과를 통해 각 무인기의 3D 최적 침투 경로 생성, 상황 변화 시 실시간 경로 재계획 수행
 * 공대지/공대공 협업 자율 임무 수행
  ** 지상 표적 공격 임무를 수행하는 무인기가 위협 구역/장애물을 회피하여 표적까지 침투·돌입하는 전술 기동을 단일 모델로 통합 학습
  ** 공중 위협으로부터 무인기를 방어하기 위해 자율 요격 기동을 수행하는 비행 지능 개발
  ** 다수 무인기가 지상 표적·상황 변화 시, 대형을 실시간으로 재배치 하여 관측 커버리지와 추적 연속성을 동시 확보

- 자율화 무인기 Physical AI 학습/실증 환경
 * 시나리오/합성데이터 생성 스택
  ** 시나리오를 자동 생성·변이하여 대량의 합성 데이터를 구축하고, 소량의 실비행 데이터와 단계적으로 혼합·정합하여 학습 데이터의 품질과 현실성을 동시에 확보
 * 자율에이전트 학습 파이프라인
  ** RL 기반 학습 알고리즘과 다목적 보상·정책 최적화, MARL 협업 학습 구성을 통합하고 MLOps 기반 재현성 도구로 전 주기를 표준화
 * Sim-to-Real 전이
  ** 시뮬레이션-실환경 간 격차를 분석·강건화하고 실측 데이터로 적응시킨 뒤 SIL/HILS로 전이 성능을 검증
 * 자율성/생존성 평가
  ** 자율성·생존성·안전성 제약 준수를 정량 평가하고 결과를 자동 분석·리포팅
 * 통합실행/오케스트레이션
  ** 시뮬레이션 학습 운영을 통제하고 의존성 기반으로 실행을 스케줄링하며 상태를 실시간 모니터링·로깅하여 임무 실행 명령을 자율 출력
