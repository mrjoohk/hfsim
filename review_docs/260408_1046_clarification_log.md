## Resolved Ambiguities
- 실행 환경은 단일 머신 기준으로 정리한다.
- 성능 목표는 고정 rollout 수보다 `60x 시간 배속`과 하드웨어 적응형 병렬화에 중점을 둔다.
- 초기 관측은 구조화 feature 중심으로 제한한다.
- 평가 우선순위는 prediction error > latent consistency > policy convergence로 확정한다.
- 멀티에이전트 기본 규모는 4기로 유지한다.

## Unresolved Items [UNRESOLVED]
